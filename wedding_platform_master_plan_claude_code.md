# Wedding Event Platform — Master Plan

결혼식/라이프 이벤트 SaaS를 MVP 없이 완성형 제품으로 바로 출시하기 위한 기준 문서다.

## 목표

1. 개인 고객이 바로 사용할 수 있는 고품질 모바일 청첩장/이벤트 페이지 제공
2. RSVP, 게스트 운영, 체크인, 메시징, 정산/리포트까지 포함한 운영 플랫폼 구축
3. 추후 웨딩홀/플래너/스튜디오/기업 이벤트로 확장 가능한 멀티테넌트 B2B 구조 확보
4. 프론트엔드 개발자가 단독 혹은 소규모 팀으로도 빠르게 개발/운영 가능한 현실적인 풀스택 기술 스택 적용
